# Smart Financial Advisor

## Project Overview
Smart Financial Advisor is a web-based financial analysis application that helps users analyze their expenses and gain financial insights.

The system allows users to upload financial data in CSV format and visualize their expenses using charts.

## Features
- Financial data upload
- Expense analysis
- Data visualization
- Simple financial insights
- Web and mobile support

## Technologies Used
- Python
- Streamlit
- React
- React Native
- Pandas

## How It Works
1. User uploads financial CSV data.
2. Streamlit backend processes the data.
3. Expenses are grouped by category.
4. Charts visualize financial spending.
5. Users receive insights about their expenses.
